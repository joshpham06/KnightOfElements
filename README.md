App Title: Knight of the Elements 

Target Audience: Children under the age of five months, and Jon 

Overview of App: Knight of the Elements is a top-down gladiator style combat survival game. 
 
Core Functionality-
Enemy Waves  
Player spawns in a small arena with basic armor, a sword, and five elements.  
Enemies spawn in waves with increasing difficulty — goblins, undead, elementals, etc.  
The knight uses elemental projectiles or melee-infused attacks to survive. 
Enemies spawn at arena edges.  
Each wave increases in speed, number, or health.  
Simple enemy AI: chase player (A*). 
Input System 
WASD movement 
Mouse to aim 
12345 to switch elements 
Elemental Abilities 
Can switch to different elements 
Each element has different abilities, rock-paper-scissors style damage 
Special attack for each element that charges with dealt damage
